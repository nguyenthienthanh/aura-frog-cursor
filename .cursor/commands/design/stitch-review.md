# /design:stitch-review

## Description

Review designs generated by Google Stitch AI. Validates against design system, accessibility standards, and responsive requirements.

## Usage

```
/design:stitch-review
/design:stitch-review [component-name]
```

## Review Checklist

- [ ] Design system compliance (colors, typography, spacing)
- [ ] Accessibility (contrast ratio, touch targets, ARIA labels)
- [ ] Responsive behavior (mobile, tablet, desktop)
- [ ] Interaction states (hover, focus, active, disabled, loading)
- [ ] Edge cases (long text, empty states, error states)
- [ ] Dark mode support (if applicable)

## Related Commands

- `/design:stitch` - Generate designs
- `/visual:test` - Visual regression testing
- `/workflow:phase-3` - UI breakdown phase
